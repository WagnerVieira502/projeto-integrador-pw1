# Projeto Integrador - PW1

Projeto de animação com sprites utilizando HTML, CSS e JavaScript.

## Funcionalidades
- Seleção de animações via sprite sheet
- Customização de skin do personagem (cores diferentes)
- Controle de nível de dificuldade do jogo

## Tecnologias
- HTML5
- CSS3
- JavaScript
- Canvas

Projeto desenvolvido para a disciplina PW1.
